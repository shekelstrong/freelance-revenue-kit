# Freelance Revenue Kit

MCP-сервер для монетизации фрилансера.

## Когда использовать

- Хочешь начать зарабатывать фрилансом с нуля
- Один канал не кормит — нужны параллельные
- Нужны деньги срочно (3 дня)
- Хочешь понять какой канал дохода приоритетнее

## 4 tools pipeline

```
старт → plan_first_revenue (7 дней)
       ↓
каналы → parallel_channels (оптимизация)
       ↓
ROI → revenue_tracker (что масштабить)
       ↓
SOS → emergency_strategy (3 дня)
```

## Алгоритм

### 1. plan_first_revenue
7-дневный план:
- День 1: 5 кворков на Kwork (минимум 400₽)
- День 2: 20 откликов на заказы
- День 3: 10 DM в TenChat/VC/Telegram
- День 4: Портфолио (3-5 проектов в GitHub)
- День 5: 20 новых откликов
- День 6: Пост в TenChat/VC
- День 7: Первый заказ (или повтор дня 5)

3 уровня urgency: critical (<14 дней runway), high (14-30), normal (30+).

### 2. parallel_channels
6 каналов с разными payout days:
- Kwork (вт/пт), FL.ru (пт), Plati (instant), LolzTeam (instant), TenChat (manual), Upwork (weekly)

Синергия:
- Kwork + Plati: Kwork = клиенты, Plati = 24/7 продажи
- Kwork + TenChat: Kwork = quick cash, TenChat = B2B
- Lolz + TenChat: Lolz = cash, TenChat = бренд

### 3. emergency_strategy
5 стратегий для 3-дневного deadline:
- Перепродажа цифровых (Plati/Lolz) — instant 3-10к₽/день
- Kwork за 400₽ × 5 = 2000₽/день
- Крипто-арбитраж — 1-5к₽/день
- AI-агент — долгосрочно, не для 3 дней
- Outreach к знакомым

### 4. revenue_tracker
ROI % + profit/hour по каналам + рекомендация.

## Pitfalls

| Ошибка | Последствие | Как избежать |
|---|---|---|
| Один канал | Kwork = 2 выплаты/нед | 3 канала параллельно |
| Ждать "идеального" заказа | 0 cash | Бери что есть |
| Соглашаться без договора | Неоплата | Используй безопасную сделку |
| Вкладывать в рекламу | Минус | Только органический outreach |
| Крипто-фьючерсы с плечом | -95% | Спот только |
| Продавать краденое | Бан | Только свои аккаунты/ключи |
| Ждать портфолио | Теряешь неделю | Сделай на коленке и иди |
